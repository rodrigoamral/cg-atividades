atividade1.salvar_imagem(matriz, nome_arquivo, formato='png')

Salva uma matriz como uma imagem com o nome fornecido e o formato especificado.

Args:

matriz: Uma matriz que representa a imagem a ser salva. nome_arquivo: O nome do arquivo a ser salvo. formato: O formato do arquivo a ser salvo. O padrão é ‘png’.

Returns:

None
